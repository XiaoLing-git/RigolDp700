# RigolDp700
 Support for Rigol DP700 Series power supplies.

## Notes:

Before you start it, please make sure you have installed the following tools

- make
- poetry
- black
- mypy
- ruff

## Commands

### Format Code

```bash
make format
```

### Type Check

```bash
make check
```

### Build Wheel

```bash
make build
```

### Install 

```bash
make install
```

### Enter Venv 

```bash
make shell
```

### Clean

```bash
make clean
```

### Commit To Github

```bash
make commit msg="comments"
```

### Test

```bash
make test
```

### debug

```bash
make debug
```
