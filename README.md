# RigolDp700
 Support for Rigol DP700 Series power supplies.

## Notes:

Before you start it, please make sure you have installed the following tools

- make
- Poetry
- black
- mypy
- ruff

## Commands

### format Code

```bash
make format
```

### Type Check

```bash
make check
```

### Build Wheel

```bash
make build
```


### Clean

```bash
make clean
```

### Commit To Github

```bash
make commit
```

### Test

```bash
make test
```
